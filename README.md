# Passive analysis

Distributed passive traffic analysis for local area networks. Made using python, tshark, MongoDB, django...

## Requirements

### Capturing
- python 3+
- Tshark 3+
- Pyshark

### Preprocessing
- python 3+
- pymongo

### Database and application
- docker
- docker-compose
