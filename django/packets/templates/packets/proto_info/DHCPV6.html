About {{ protocol }}

This protocol can give information such as:

  • Manufacturer: The first bits of the MAC of the devices are related to the device manufacturer. If the user did not change its MAC, the MAC address itself will reveal the device manufacturer.
  • IPv6 status: This protocol indicates that IPv6 is enabled on the device.
  • Operating System: This protocol uses the IP protocol. OSs use different TTL. For example, linux might use a TTL of 64 meanwhile windows might use a TTL of 128.
  • Application port: This protocol uses the UDP protocol. The value srcport of the UDP protocol contains the port used by the application.
  • Vendor class: The field option_vendor_class_id of the DHCP protocol indicates the vendor class of the device
