About {{ protocol }}

This protocol can give information such as:

  • Manufacturer: The first bits of the MAC of the devices are related to the device manufacturer. If the user did not change its MAC, the MAC address itself will reveal the device manufacturer.
  • IP address: This protocol uses the IP layer, so you can get the device IP address.
  • Type of device: The protocol BJNP is used by Canon printers and scanners.
  • Device function: The protocol BJNP protocol uses different ports for different purposes. The four ports/purposes are 8610 for advertising its IP address, 8611 for printing, 8612 for scanning and 8613 for sending and receiving faxes.
